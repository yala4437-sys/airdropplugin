# AirDrop Plugin

Автоматический плагин воздушных дропов для Paper 1.20.1

## Возможности

- 🎁 **Автоматические дропы** каждые 60 минут
- 🌟 **Редкие и обычные дропы** с разными ресурсами
- 📍 **Случайные локации RTP** для каждого дропа
- ⏱️ **Таймер 5 минут** перед взрывом
- 💣 **Взрыв и разлёт ресурсов** по окончании таймера
- 🔒 **Неломаемая бочка** до взрыва
- 📢 **Оповещения в чат** с координатами

## Установка

1. Скомпилируйте плагин:
```bash
mvn clean package
```

2. Скопируйте JAR файл в папку `plugins/` вашего сервера

3. Перезагрузите сервер

## Команды

- `/airdrop` - Создать дроп вручную (требует `airdrop.admin`)
- `/airdropstatus` - Проверить количество активных дропов

## Конфигурация

Вы можете изменить параметры в `AirDropManager.java`:

- `INTERVAL_MINUTES` - Интервал между автоматическими дропами (по умолчанию 60)
- `TIMER_SECONDS` - Время до взрыва в секундах (по умолчанию 300 = 5 минут)
- `RARE_CHANCE` - Шанс редкого дропа (по умолчанию 0.20 = 20%)
- `RTP_RADIUS` - Радиус RTP в блоках (по умолчанию 1000)

## Редкие дропы

**Обычный дроп:**
- Железо x32
- Золото x16
- Медь x24
- Уголь x32
- Изумруды x8

**Редкий дроп:**
- Алмазы x16
- Изумруды x24
- Золото x32
- Нетерит x4
- Зачарованные золотые яблоки x4
